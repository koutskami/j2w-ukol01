O mně

Vystudovala jsem bohemistiku a informační vědy, zajímám se o populární kulturu a využití digitálních technologií (nejen) v humanitních vědách. Během studia jsem také začala objevovat datovou analýzu a testování díky kurzům Czechitas.

Po skončení studia jsem získala pozici manuální testerky v týmu, který vyvíjí bankovní API a služby Bank iD. Momentálně se seznamuji s programováním v Javě a chtěla bych se naučit testování automatizovat.
